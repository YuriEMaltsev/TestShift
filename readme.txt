  Программа Parser Digital
  	Stream Parser
    Назначение
  	Программа выборки из исходных файлов строк со строками,
  	содержащие Long и Double
    Автор
  	Юрий Мальцев
    Дата релиза
  	11.02.2026

Ключи:

   -o  путь к целевой директории (умолчание текущая директория)
     
   -a дописывать в существующие файлы

   -f полный формат отчета
   
   -b десятичный размер буфера потока чтения (умолчание 65535 байт)

   -p префикс имен файлов  

Пример командной строки Main -o c:\tmp -p result -a c:\tmp\list2.txt c:\tmp\list3.txt c:\tmp\list1.txt  -s -f  

Версия  java 11.0.15
Среда разработки IntelliJ IDEA 2025.3
